1) Open an "x64 Native Tools Command Prompt for VS 2022"
2) In the src folder, run buildLib.bat
3) Copy sfxObjectDump.lib to this folder
4) Run build.bat
5) Run the viewer with $ SuperFXViewer [path to .rom]

Press left and right arrows to cycle through all of the models

Press 'd' to change the distance lighting attenuation.
